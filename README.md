# MetAFT: Meta-Programming for Approximation Fixpoint Theory

A system for semantics of approximation fixpoint theory based semantics for normal logic programs using meta-programming in answer set programming.

## Overview

MetAFT implements approximation fixpoint theory (AFT) semantics for normal logic programs through meta-programming techniques in Answer Set Programming (ASP). This project provides a framework for computing and analyzing fixpoint-based semantics for normal logic programs with negation.

## Contents

- All semantics can be used using the tool in ``tool``. Detailed instructions are found in that folder as well.
- Experiments are kept in ``benchmarks``.
